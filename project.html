Projects

Cyberbullying Detection using Machine Learning

Developed a web application using Python and Django that leverages ML and DL(RNN,LSTM) techniques to identify and predict cyberbullying tweets. The system is integrated with NLP tools for text analysis and classification. It provides a real-time alert feature by sending email notifications to parents when cyberbullying is detected.

Credit Risk Analysis

Developed a system to predict creditworthiness using Python. The project includes EDA & multiple ML models. The model was evaluated using various metrics and deployed as a user-friendly application with a Tkinter-based interface.

Strong password generator based on user input

Developed a user-friendly application using Python and Tkinter, offering secure password generation and encryption. The system utilizes memory-based storage instead of databases, minimizing hacking risks. Key features include a Tkinter-based GUI, secure memory management, and options to receive passwords via email, WhatsApp, and SMS.

A study on 5G technologies and it's applications in telecommunications

5G, the fifth generation of mobile network technology, represents a transformative leap in telecommunications. It offers ultra-fast data speeds, low latency, massive device connectivity, and enhanced network reliability. This study explores the core principles behind 5G technology, including its infrastructure, spectrum usage, and architecture.
